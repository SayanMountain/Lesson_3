1. Тема классы.
Классы - базовая единица всех программ в Java

Какую функцию выполняет класс? Поскольку Java программы состоят из набора классов, где
каждый класс определяет уникальный тип объектов, то есть нвоый тип данных
Если говорить лаконично, то классы позволяют создать новый тип данных для объекта внутри класса

Каждый класс содержит методы.
Если отвелечься, то  что функции исходят из функциональных ЯП
А методы свойственны ООП ЯП

Название файла в котором содержится класс должно совпадать с его именем
f.e. Door door

Для лучшего запоминания стоит провести аналогию, в которой в большинстве случаев
класс является моделью объекта из реального мира (Машина, Кот, Человек)
Класс - фирма КИА
Объект - модель Оптима
Объект - модель Соренто

Переходим в форму создания класса, см. lesson_3_lection
